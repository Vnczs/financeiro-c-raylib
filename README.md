#  App de Controle Financeiro em C (Raylib)

Este é um projeto de gerenciador financeiro pessoal desenvolvido em **Linguagem C** utilizando a biblioteca gráfica **Raylib**. O foco do projeto foi criar uma interface intuitiva para dispositivos móveis, aplicando conceitos de persistência de dados e lógica de programação.

## Funcionalidades
- Interface gráfica com layout vertical (Mobile Friendly).
- Entrada de valores com formatação de moeda brasileira (R$).
- Histórico automático das últimas transações.
- Persistência de dados em arquivo `.txt`.

## Tecnologias Utilizadas
- **Linguagem C**
- **Biblioteca Raylib** (Interface Gráfica)
- **Dev-C++** (Ambiente de Desenvolvimento)
